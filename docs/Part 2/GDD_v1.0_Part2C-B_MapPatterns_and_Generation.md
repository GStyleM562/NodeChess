# 1. Design Philosophy

Maps should be reusable frameworks.

Maps should be classified.

Maps should have recognizable identities.

Different archetypes should support different deck strategies.

Maps should encourage:

• Combat

• Positioning

• Spawn Control

• Buff Node Contesting

• Goal Defense

• Tactical Choices
