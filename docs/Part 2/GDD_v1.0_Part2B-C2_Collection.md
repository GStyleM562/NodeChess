# 1. Collection Philosophy

The Collection System exists to encourage:

• Ownership

• Personalization

• Progression

• Completion

• Discovery

• Experimentation

The Collection should feel rewarding.

The Collection should feel expandable.

The Collection should remain easy to navigate.
