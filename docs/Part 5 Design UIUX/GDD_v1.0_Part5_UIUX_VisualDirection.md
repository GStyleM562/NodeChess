# 1. UI Philosophy

The game should feel like:

Pokemon Duel
+
Clash Royale
+
Modern Mobile UX

The game should prioritize:

• Fast readability

• One-handed usability

• Immediate understanding

• Premium feeling

• Minimal clutter

• Responsive interactions

• Easy deck management

The objective is:

A player opening the game for the first time should understand where to click within 5 seconds.
